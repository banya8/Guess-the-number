Guess-the Number

📌 Overview

it is fun game to play in your downtime. Where you have to guess a four digit number. R indicate the digit and the position is correct. Y refers that the digit is in the number but in the wrong place. and finally B refers  the digit is not a part of the number.

🛠️ Technologies Used

Python 

💡 Usage

Open code and dive in to find out the number.
